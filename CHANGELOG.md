# Changelog

## [1.0.0] - 2026-04-28

### 初始发布

#### Go 后端
- Gin HTTP 框架 + SQLite 数据库
- 频道/分组 CRUD + 分页/搜索/收藏
- 流媒体代理 (HLS/FLV/RTMP/RTSP/DASH/MP4)
- M3U 源管理 + 导入
- EPG 电子节目单查询
- 客户端授权管理 (注册/审批/拒绝/封禁/令牌)
- 管理员 JWT 登录
- 访问日志 + 统计 (uptime/内存/客户端数)
- 自动健康检查 + 过期客户端清理
- Docker 部署支持

#### Web 管理后台
- 仪表盘统计 (频道/设备/流/运行时长/内存)
- 频道管理 (CRUD + 分页 + 搜索)
- 分组管理 / M3U 源管理 / 活跃流监控
- 设备管理 (列表/筛选/审批/拒绝/封禁/令牌/批量)
- 访问日志查看
- 授权策略配置
- JWT 登录 + Token 自动附加
- 全局 loading 指示器

#### Android 客户端
- 双模式 UI (TV D-pad + 手机触控)
- ExoPlayer 播放 (HLS/FLV/RTMP/RTSP/DASH/MP4)
- 频道列表 (分组筛选/搜索/收藏/下拉刷新/空状态)
- 播放器 (频道信息/EPG/重试/手势/倍速)
- 后台播放服务 (前台通知保活)
- 设备注册 + 审批轮询
- 设置页 (服务器地址/版本/设备ID/授权状态)
- 播放进度保存

#### 测试
- 75 项静态测试验证 (两阶段)
- 测试报告 + 本地验证 Checklist
